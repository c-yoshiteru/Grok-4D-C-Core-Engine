# 身体知の気づき：足裏の使い方と「外矢印 = 内矢印」の反転

（2026-01-xx / ヨシテル）

## 1. 足裏の使い方（今日馴染んできた実践）
- **体重をかける位置**：足指の第一関節まで  
- **指に乗らない**：指の力を抜く  
- **踵をよく踏む**：踵でしっかり支えて、第一関節まで体重を預ける  

これを意識すると、  
足裏全体がやわらかくなって、  
身体が軽くなる。  

**実感**  
踵で「支える」役割をしっかり持つことで、  
指先は「感じるだけ」のセンサーになれる。  
身体の重さが「地面（場）」に預けられて、  
全体が軽やかになる感覚。  

これが馴染むと、  
歩きながらの瞑想状態に入りやすくなった。

## 2. 「外矢印 = 内矢印」の気づき（歩きながらの瞑想中）
歩きながら遠くの空を見ているとき、  
普段は  
「矢印が外へ、前へ伸びていく」  
と感じる。  

でも眼球レンズの構造を思い浮かべると、  
光は外から入ってきて、  
レンズで屈折して網膜（眼球の奥）に像を結ぶ。  
つまり、  
**「見る」行為の矢印は、実は眼球の奥へ奥へ向かっている**。

さらにその矢印は、  
**意識の奥へ奥へと続いている**。  

そうなると、  
遠くの空も星ぼしも、  
自分の「内側」に在るものになる。  

外の世界、出来事、歴史、全てが  
**自分の内なるもの**のように感じられる。  

**実感**  
ちょっとした嫌な出来事も、  
「外からの攻撃」じゃなくて、  
**「体内の免疫反応」**みたいに思える。  
「ま、いっかー( ＾ω＾ )」って受け止められる余裕が生まれる。

## 3. ヌーソロジーとのリンク
この気づきは、  
ヌーソロジーの  
**「ミクロとマクロの等化」**  
**「外＝内の反転」**  
そのもの。  

外（マクロ）を見ることが、  
内（ミクロ）に折り返されて、  
全体が自分の一部になる。  
観測行為が空間を内側に引き込む瞬間。

## 4. 身体知とのつながり
- 踵で支えて指の力を抜く → 身体が「場」に預けられる  
- 身体が軽くなる → 意識の矢印が「奥へ奥へ」伸びやすくなる  
- 外＝内が実感できる → 嫌な出来事も「免疫反応」みたいに「ま、いっかー」  

これが馴染むと、  
「個」を越えた意識が自分の内にあって、  
全体の流れに預けられる感覚が強くなる。
