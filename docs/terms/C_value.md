# C-value.md

## C値（C-value） - 基本版（4段階）

**Invariant motivational signal extracted from contradictory vectors**  
→ 矛盾ベクトル群から抽出される「不変の動機信号」

### 範囲と段階（4段階モデル）
- 0.0〜0.4: CHAOS（軸なし、ノイズ支配）  
- 0.2〜0.6: INVERT（軸形成中、視点反転）  
- 0.5〜0.8: SYNC（動的平衡、関係性観測）  
- 0.8〜1.0: UNITY（軸統合、ノイズ最小）
